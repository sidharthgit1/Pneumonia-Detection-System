# Dataset

This project uses a chest X-ray image dataset for the classification of respiratory conditions.

## Classes

The dataset contains three categories:

* Pneumonia
* COVID-19
* Normal

## Dataset Source

The dataset was obtained from Kaggle and consists of chest X-ray images collected for medical image classification research and educational purposes.

## Dataset Structure

```text
dataset/
├── Pneumonia/
├── COVID/
└── Normal/
```

## Note

The dataset is not included in this repository due to GitHub storage limitations and licensing considerations.

To run this project, download the dataset from the original Kaggle source and place the images in the `dataset` directory following the structure shown above.

## Purpose

The dataset is used to train and evaluate a deep learning model for automated chest X-ray classification into Pneumonia, COVID-19, and Normal categories.
